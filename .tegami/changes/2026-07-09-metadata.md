### Ship package metadata to npm

`repository`, `homepage`, `bugs` and keywords were added to the manifest but never
reached the registry — 0.1.0 was published before they existed. The npm page now
links back to the source, and the package is discoverable by keyword.

### Build with TypeScript 6

The published bundle is now produced by TypeScript 6.0.3. `dist` no longer ships
`.js.map` files: the maps referenced sources under `src/`, which `files: ["dist"]`
never published, so they pointed at nothing.
